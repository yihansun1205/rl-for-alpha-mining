# RL for Alpha Mining

A research repository for literature review, technical notes, reproduction progress, and experimental studies on reinforcement learning for formulaic alpha mining.

## Repository Structure

- `docs/surveys/`  
  Literature surveys and review drafts

- `docs/technical_notes/`  
  Technical notes, model/module breakdowns, and source-code understanding

- `docs/code_maps/`  
  Code-reading maps for representative repositories

- `docs/plans/`  
  Writing plans, reproduction plans, and technical documentation plans

- `reproductions/`  
  Reproduction progress for representative papers

- `experiments/`  
  Experiment logs and ablation notes

- `figures/`  
  Diagrams and visual assets

- `references/`  
  Paper lists and useful links

- `drafts/`  
  Ongoing paper drafts and writing materials

## Current Focus

- RL for formulaic alpha mining
- AlphaGen / AlphaForge / AlphaQCM
- Literature review and technical documentation
- Reproduction planning and experiment tracking
